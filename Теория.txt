Гит команды

git init - создание локального репозиторич
git status - проверка статусов файлов
git add имя_файла.формат - добавление файла в отслеживаемые
git rm --cached имя_файла.формат - отключение слежения гита за файлом
git rm -f --cached имя_файла.формат - отключение слежения гита за файлом
git log - посмотреть историю коммитов
git log -1 -p - посмотреть последний коммит
git commit -m "Имя коммита" - делаем коммит в репозиторий

git checkout хэш -  откат до определенного коммита
git checkout master - вернуться обратно на ветку мастера

для связи локального репозитория с тем что на гит хабе
git remote add origin https://github.com/TheEsus/MyBook-by-Eugene-Evtushenko-.git - создает связь
git push -u origin main - пуш данных в репозиторий на гите

клонирование гит репозитория
git clone ссылка - клонирует репозиторий на комп с гит сервера

для апдейта
 git pull - получаем обнову

для слияния веток
git merge имя_ветки_с_которой_мерджим
